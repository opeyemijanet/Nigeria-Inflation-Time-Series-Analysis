Nigeria Inflation Time-Series Analysis (2016–2024)

This project analyzes Nigeria’s CPI data to examine structural inflation trends, volatility, and urban–rural divergence.

Tools Used
- Python (Pandas, NumPy, Matplotlib)
- Time-Series Calculations (YoY, MoM)
- Rolling Volatility Analysis
- Cross-State Dispersion Metrics
- Google Looker Studio Dashboard

Data Source
Nigeria CPI dataset via Open Data for Africa.

Key Focus
- Inflation Regime Shift
- Volatility Dynamics
- Urban vs Rural Inflation Gap
- Structural Dispersion Across States
